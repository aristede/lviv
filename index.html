lviv by aristede

stack-based RPN functional programming language developed at kwantam/liv

This fork is for testing code and writing documentation with the hope of eventually contributing to kwantam/lviv repo.

Dependencies

gambit-C

Install

install Gambit C git clone https://github.com/aristede/lviv.git Then to run as REPL: cd lviv/src gsi lviv.scm
